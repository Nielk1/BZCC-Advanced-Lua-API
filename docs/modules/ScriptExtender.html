RadarDisable (type)
Disable radar type. If this radar type is active, it will automatically switch to the next active type or none

Parameters:

  • type RadarType Radar Type to disable
RadarEnable (type[, activate])
Enable radar type. If no radar types are active or activation is forced this radar will activate as well as enable

Parameters:

  • type RadarType Radar Type to disable
  • activate bool Force activate this radar (optional)

RadarDisable (type)
Disable radar type. If this radar type is active, it will automatically switch to the next active type or none

Parameters:

  • type RadarType Radar Type to disable

Usage:

    RadarDisable(RadarType.RADARTYPE_MAP);
RadarEnable (type[, activate])
Enable radar type. If no radar types are active or activation is forced this radar will activate as well as enable

Parameters:

  • type RadarType Radar Type to disable
  • activate bool Force activate this radar (optional)

Usage:

  • RadarEnable(RadarType.RADARTYPE_MAP);
  • RadarEnable(RadarType.RADARTYPE_MAP, true);
RadarEnabled (type)
Is radar type enabled?

Parameters:

  • type RadarType Radar Type to check

Returns:

    bool if the radar type is enabled

Usage:

    local enabled = RadarEnabled(RadarType.RADARTYPE_MAP);
RadarActive (type)
Is radar type active?

Parameters:

  • type RadarType Radar Type to check

Returns:

    bool if the radar type is active

Usage:

    local active = RadarActive(RadarType.RADARTYPE_MAP);

GameObject

SetStatusImage (handle, texture)
Change the status image of an object. This image, conventionally called the "wireframe" in BZ2, is the image above the weapon panel. This code exploits the logic used to change it for pilot pack weapons.

Parameters:

  • handle Handle GameObject handle
  • texture string Texture filename

Usage:

    SetStatusImage(playerHandle, 'wire_iggren3.dds');
SetLockdown (handle, turns)
Set the lockdown state on an object. Apply the EMP Stream affect to an object.

Parameters:

  • handle Handle GameObject handle
  • turns number game turns to apply affect

Usage:

    SetLockdown(objectHandle, GetTPS() * 10);
GetLockdown (handle)
Get the lockdown state on an object. Get the EMP Stream affect turns remaining.

Parameters:

  • handle Handle GameObject handle

Returns:

    int number turns game turns left of affect

Usage:

    local turns = GetLockdown(objectHandle);

SetRadarColor (handle, color)
Set base radar color of object. This sets the radar color on stock radars for an object and auto-generates the 2nd color. The game will override this when a color change event occurs, such as hopping out.

Parameters:

  • handle Handle GameObject handle
  • color ColorInt Color value

Usage:

    SetRadarColor(objectHandle, 0xff0000);
GetRadarColor (handle)
Get base radar color of object. This gets the radar color on stock radars for an object.

Parameters:

  • handle Handle GameObject handle

Returns:

    ColorInt Color value

Usage:

    local color = GetRadarColor(objectHandle);
ResetRadarColor (handle)
Reset radar color of object. This restores the game default radar color to an object.

Parameters:

  • handle Handle GameObject handle

Returns:

    ColorInt Color value

Usage:

    local color = GetRadarColor(objectHandle);
GetRadarColorWithState (handle)
Get radar color of object considers its alive state. Objects that are dead have a darker color, though this appears broken in BZCC it is still in the code and may be used in other cases.

Parameters:

  • handle Handle GameObject handle

Returns:

    ColorInt Color value

Usage:

    local color = GetRadarColorWithState(objectHandle);

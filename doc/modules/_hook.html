hook.CallAllPassReturn (event, ...)
Calls hooks associated with the hook name passing each return to the next. Hooked functions may return 2 values. The first will be passed to the next as the last paramater. The second, if true, will stop the execution of hooks.

Parameters:

  • event Event to be hooked (string)
  • ... Paramaters passed to every hooked function

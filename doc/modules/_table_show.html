table.show (t, name, indent)
Convert table to string representation.

Heavily based on "Saving tables with cycles", PIL2, p. 113. Formats tables with cycles recursively to any depth. The output is returned as a string. References to other tables are shown as values. Self references are indicated.

The string returned is "Lua code", which can be procesed (in the case in which indent is composed by spaces or "--").

Userdata and function keys and values are shown as strings, which logically are exactly not equivalent to the original code.

Parameters:

  • t the table
  • name name of the table (optional)
  • indent first indentation (optional)

Usage:

    print(table.show(t, "t"))   -- a typical use
